# 易因 · 世界分析引擎 — 更新日志

## v3.1.0 (2026-05-12) — Round 3：深度重构与数据层修复

### 核心修复
- **修复 guaData 重复 key 问题**：原数据中 `qian`、`bi`、`jian` 等卦象出现多次重复 key，导致后面的数据覆盖前面的，64卦数据不完整。现已为所有重复卦分配唯一 key（如 `bi_gua`、`bi2`、`jian2`、`jian3`、`qian2` 等），确保 64 卦数据全部完整保留。
- **修复 guaPatterns 重复 key**：与 guaData 对应，guaPatterns 也同步去重，确保关键词匹配算法能正确命中所有 64 卦。
- **修复 pratityaData 缺少 color 字段**：为十二因缘 12 个节点全部补充 `color` 字段（`node-avidya` 到 `node-jaramarana`），修复节点显示异常。

### 架构升级
- **CSS 独立拆分**：将原先内嵌在 HTML 中的 ~2350 行 CSS 提取为独立 `styles.css` 文件（~58KB），提升可维护性和浏览器缓存效率。
- **JS 独立拆分**：将原先内嵌在 HTML 中的 ~1600 行 JavaScript 提取为独立 `app.js` 文件（~109KB），HTML 仅保留结构，实现关注点分离。
- **index.html 精简**：从 4122 行 / 206KB 精简为 437 行 / 26KB，仅包含 DOM 结构和外链引用。

### 存储层升级
- **IndexedDB 替换 localStorage**：历史记录、收藏、对比列表全面迁移至 IndexedDB（`yiyin_db`），支持更大容量（>5MB）、更好查询性能、结构化索引（按时间、收藏状态）。
- 新增 `dbGetAll`、`dbPut`、`dbDelete`、`dbClear` 等通用 IndexedDB 工具函数。
- 保留 localStorage 仅用于对比列表的轻量状态同步。

### 新增功能
- **图片导出（DOM-to-Image）**：使用纯原生 Canvas 2D API 实现分析结果导出为 PNG 图片，无需外部依赖。支持高分辨率（2x 缩放）渲染，包含卦象、因缘、交叉分析和干预建议的完整卡片。
- **时间起卦动爻与变卦**：新增 `determineMovingYao` 和 `calculateChangedGua` 函数，根据输入文本哈希计算动爻，并推演出变卦，在结果中展示「变卦启示」。
- **六爻可视化**：分析结果中新增六爻展示，包含阴阳爻条、爻辞，动爻高亮标记。
- **标签系统增强**：支持 Enter/逗号添加标签、Backspace 删除标签、标签在历史和收藏卡片中展示。

### 交互优化
- 命令面板（⌘K）新增「导出图片」命令（⌘⇧I）。
- 时间起卦支持「采用此卦」按钮。
- Toast 通知支持 info/success/warning/error 四种类型。
- 所有视图（历史、收藏、对比、六十四卦、十二因缘）支持键盘导航和快捷键。

### PWA 维护
- `sw.js` 和 `manifest.json` 保持不变，但已确认缓存策略兼容新文件结构。
