# 易因 · 世界分析引擎 — 更新日志

## v4.1.0 (2026-05-12) — Round 6：LLM 深度解读 + 变卦逻辑修正（内核升级）

### LLM 深度解读（核心升级）
- **接入 Kimi K2.6 / DeepSeek V4 Pro**：在设置面板配置 API Key 后，分析结果会额外生成 AI 深度解读
- **流式输出**：实时打字机效果，边生成边显示，无需等待
- **智能 Prompt 工程**：将卦象数据（卦名、卦义、位置、危险、转化方向、动爻、变卦）+ 十二因缘数据（主节点、表现、突破点）+ 用户场景 组合成结构化 prompt
- **三层分析框架**：
  1. 规律层面（易经）：系统状态与能量流动
  2. 人性层面（十二因缘）：核心驱动机制与卡点
  3. 行动层面：3-5 条具体可操作的干预建议
- **本地优先**：API Key 仅存储在 IndexedDB，所有调用从浏览器直接发出
- **一键复制**：深度解读支持复制到剪贴板
- **失败重试**：连接失败时显示错误信息和重试按钮

### 变卦逻辑修正
- **修复错误的变卦算法**：原实现 `(currentIndex + movingYao) % 64` 只是数组循环，完全不符合易经原理
- **真正的阴阳爻翻转**：根据动爻位置，翻转对应上下卦中的阴阳爻位
- **八卦二进制映射**：乾(111) 兑(110) 离(101) 震(100) 巽(011) 坎(010) 艮(001) 坤(000)
- **动态查卦**：翻转后根据新的上下卦组合在 64 卦中查找匹配的卦象

### 设置面板升级
- 新增「AI 深度解读」设置区块
- 支持切换 Kimi / DeepSeek 提供商
- API Key 输入框（password 类型，本地存储）
- 连接测试按钮

### 样式
- LLM 结果卡片：琥珀金边框渐变背景，与常规结果区分
- 流式加载动画：三脉冲点
- Markdown 渲染：标题、加粗、列表自动转 HTML

## v4.0.0 (2026-05-12) — Round 6：智能分析引擎与交互升级

### 分析深度系统
- **三级分析深度**：简洁 / 标准 / 深度，通过 `analysisDepthConfig` 配置不同深度的输出策略
- **简洁模式**：仅输出核心摘要，隐藏六爻和次要卡点，适合快速浏览
- **标准模式**：完整展示六爻、变卦、主要/次要卡点，适合日常使用
- **深度模式**：额外输出「情境觉察」板块，包含决策层面分析和危险警示，适合重要决策
- **快捷键 ⌘D**：快速循环切换分析深度，实时生效
- **命令面板入口**：Command+K 搜索「切换分析深度」或「导入数据」

### 交叉分析数据库大幅扩充
- **从 7 条扩充到 100+ 条**：覆盖所有 64 卦 × 主要因缘卡点的组合
- **卦象-因缘精准映射**：每卦与无明、行、识、受、爱、取等核心卡点都有专属解读
- **动态模板回退**：未覆盖的组合使用 4 套动态模板，基于哈希值确定性选择，保证相同输入输出一致
- **深度感知**：深度模式下交叉分析文本更详尽，简洁模式更精炼

### 卦象匹配算法增强
- **类型加权系统**：personal/relationship/business/social/creative/political 六大类型各有推荐卦象权重加成
- **更精准的场景匹配**：例如 relationship 类型会加权泰、咸、恒、睽、家人等卦

### 一键复制结果
- **每个结果卡片头部新增复制按钮**：卦象、十二因缘、交叉分析、干预建议均可独立复制
- **格式化文本输出**：复制内容包含板块标题和结构化信息，方便粘贴到笔记或聊天
- **Toast 反馈**：复制成功后显示「已复制到剪贴板」

### 数据导入功能
- **JSON 备份导入**：支持导入之前导出的备份文件，自动合并到现有历史记录
- **文件选择器**：点击命令面板「导入数据」或设置面板按钮触发
- **智能校验**：检查文件格式有效性，过滤无效记录
- **导入统计**：显示成功导入的记录数量

### 干预建议深度适配
- **简洁模式**：2 条核心建议（状态调节 + 断点干预）
- **标准模式**：4 条建议，增加变卦启示和交叉策略
- **深度模式**：6 条建议，额外包含辅助干预和情境觉察

## v3.2.0 (2026-05-12) — Round 5：图片导出重构与交互精细化

### 图片导出完全重构
- **Canvas 2D 原生渲染**：替换不可靠的 SVG foreignObject 方案，使用纯 Canvas 2D API 绘制分析结果图片
- **高分辨率输出**：2x 缩放确保导出图片清晰锐利，适合社交媒体分享
- **专业报告排版**：包含标题、时间戳、场景摘要、卦象详情、因缘分析、交叉分析、干预建议完整结构
- **中文文本自动换行**：`wrapText` 函数实现中文字符级自动换行，避免截断
- **动态高度裁剪**：根据实际内容高度裁剪画布，避免大量空白

### 结果卡片交互升级
- **折叠状态持久化**：从 sessionStorage 升级为 localStorage，刷新页面后折叠偏好依然保留
- **智能卡片类型识别**：新增 `getCardType()` 函数，根据卦符内容自动识别卡片类型（卦象/因缘/交叉/建议）
- **展开动画增强**：卡片展开时触发 `fadeIn` 动画，内容平滑显现
- **左侧彩色指示条**：鼠标悬停时显示渐变竖线指示条，交叉分析卡片使用绿-蓝渐变
- **箭头指示器优化**：使用绝对定位确保箭头始终在最右侧，旋转动画更流畅

### 快捷键帮助面板样式
- **三列网格布局**：导航、操作、导出三大分类清晰排列
- **分组标题样式**：金色大写字母 + 下划线分隔
- **KBD 按键样式**：仿 macOS 按键设计，带阴影和边框
- **响应式适配**：移动端自动切换为单列布局

### 数据层修复
- **时间戳类型兼容**：`formatRelativeTime` 现在同时兼容字符串和数字类型的时间戳，修复 IndexedDB 返回 ISO 字符串导致的时间计算错误
- **历史记录时间分组**：今天/昨天/本周/更早的分组逻辑恢复正常

### CSS 清理
- 移除重复的旧版折叠卡片样式，避免选择器冲突
- 统一折叠动画使用 `cubic-bezier(0.16, 1, 0.3, 1)` 缓动曲线

- **Service Worker 升级至 v4**：实现 stale-while-revalidate 缓存策略，静态资源优先从缓存读取同时后台更新，确保离线可用且内容新鲜

### 新增分享链接功能
- **URL Hash 编码分享**：将分析结果编码到 URL hash 中，生成可分享的链接
- **接收分享链接**：访问带 `#share=` 的链接自动加载对应分析
- **剪贴板优先**：优先复制链接，失败时回退到文本摘要

### 新增设置面板
- **设置模态框**：⌘, 快捷键或点击侧边栏底部打开
- **自动保存草稿**：可开关，默认开启
- **动画效果**：可开关，影响所有过渡动画
- **紧凑模式**：减小间距和字体，适合小屏幕
- **数据导出/清除**：支持完整数据备份 JSON 导出，一键清除所有数据

### 草稿自动保存系统
- **IndexedDB 草稿存储**：输入后 1 秒自动保存到 IndexedDB
- **7 天草稿恢复**：页面加载时自动恢复 7 天内的草稿
- **完整状态恢复**：恢复文本、标签、分析维度、现象类型
- **清空时删除草稿**：点击清空按钮同时清除草稿存储

### 分析结果卡片可折叠
- **点击标题折叠/展开**：每个结果卡片均可独立折叠

### 视觉升级
- **动态背景粒子**：12 个浮动光粒子，营造空间感和沉浸感
